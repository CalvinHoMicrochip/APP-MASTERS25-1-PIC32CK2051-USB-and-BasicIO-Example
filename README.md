# APP-MASTERS25-1-PIC32CK2051 USB、ADC and I2C 的範例程式
  ## 本範例程式提供以下周邊的範例
  *  USB : Full Speed USB，規劃為 CDC Class Device
  *  ADC : 以 AN07 連接至 APP-MASTERS25-1 上的 VR
  *  SERCOM : 規劃 SERCOM5 為 I2C Master Mode 來操作 OLED Display
